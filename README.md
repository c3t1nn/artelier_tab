## Overview
Transform your new tab into an art gallery, featuring masterpieces from the Art Institute of Chicago.

## Installation
**Firefox Add-ons**
- Visit Firefox Add-ons
- Search "artelier:tab"
- Add to Firefox

**Manual Install**
- Download latest .xpi
- Drop into Firefox
- Accept permissions

## Features
**Artwork Engine**
- Random masterpieces
- High-quality display
- Museum collection

**Theme System**
- Day/night modes
- Location-aware
- Auto-switching

**Color Matrix**
- Palette extraction
- Real-time process
- Dynamic themes

**Interface**
- Clean design
- Material style
- Responsive

## System Requirements
- Firefox 57.0+
- Internet connection
