Transform your boring tabs into an
art gallery featuring masterpieces
from the Art Institute of Chicago.
[aesthetic][minimal]

+--------------------------------+
|         INSTALLATION           |
+--------------------------------+

[FIREFOX ADD-ONS]
> Visit Firefox Add-ons
> Search "artelier:tab"
> Click Add to Firefox
> Enjoy the art~

[MANUAL INSTALL]
> Get latest .xpi file
> Drop into Firefox
> Accept permissions
> Experience art~
